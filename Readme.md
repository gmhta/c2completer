# Shell Completer for Camus2

## TL;DR

``` shell
./c2completer -i
source ~/.bashrc # or zsh or fish
c2 <tab><tab>
```

## Install

https://github.com/gmhta/c2completer/releases

## Build

```
git clone https://github.com/gmhta/c2completer.git
cd c2completer
go install
```
